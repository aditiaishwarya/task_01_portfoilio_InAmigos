# 🌐 Aditi Aishwarya Portfolio

A modern and responsive developer portfolio built with React, TypeScript, and Tailwind CSS. Designed to showcase my projects, skills, education, and professional journey in an interactive and visually appealing way.

## 🚀 Features

- Responsive design for all devices
- Dark/Light theme support
- Smooth scrolling navigation
- Interactive project showcase
- Skills and technology section
- Education timeline
- Contact section
- Modern UI with animations
- Optimized performance

## 🛠️ Tech Stack

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Navigate to the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
npm install react react-dom npm install -D @types/react @types/react-dom
npm install @rollup/rollup-win32-x64-msvc --save-dev
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
